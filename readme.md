Internetowy quiz, aplikacja na przedmiot WWW.

/server - backend 
/public - aplikacja frontendowa

komendy dla /server:
 - kompilacja: $npm run-script compile
 - tworzenie bazy: $npm run-script createdb
 - uruchomienie: $npm run-script run
 - testy: $npm run-script test

komendy dla /public:
 - kompilacja: $npm run-script compile
